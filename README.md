# kiz委员会约定

kiz委员会运行在kiz委员会约定之上

kiz委员会在委员人数大于等于10人时开始运行

kiz委员会设委员长三名，委员若干

- 结构

    - 决策委员会
    - 宣传与发布委员会
    - 财务委员会
    - 文档编撰委员会
    - 专家联络处
    - 贡献者会议

- 文档编撰委员会

对于文档的编撰应该支付给贡献者一定的报酬

- 核心开发者

当你为委员会的项目修改了重大Bug, 或者添加了许多功能, 你可以成为kiz的核心开发者

- 元老

退休的委员可以成为元老

- 决策权

超过2/3的委员在场则作出的决策有效

**除了宣布委员会解散**需要8/10的委员投出赞成票之外


其他决策只需要6/10的委员投出赞成票

如果赞成票数与否决票数相差小于总票数的2/10, 则该决策无效, 需要进行第二次决策或者询问专家或元老的意见

**原则：少数服从多数, 众人的事由众人决定, 众人的事由众人商量**

    - 删除仓库
    - 删除项目
    - 开除委员
    - 宣布委员会解散
    - 修改kiz委员会约定
    - 决定Todo与发展规划
    - 通过议案
    - 否决议案
    - 通过合作申请

- 委员权利

    - 创建仓库
    - 创建项目
    - 提出议案
    - 邀请别人加入委员会
    - 竞选委员长(6个月选一次)

- 财务

财务委员会应当把每项收入和支出记在账本上

排除必要支出外, 剩余收入可以给委员和贡献者发福利

**严禁贪污**
